Whitepaper Abstract

Smart Data

What if data could do more for you? We use data to share photos, send emails and connect with friends, but data by itself is relatively powerless. Thought Network is creating a whole new technology based on Smart Data, a new data structure that consists of any piece of ordinary data surrounded by software logic that allows it to do things. With Smart Data, we can usher in a whole new paradigm of active data, where we can do the things we want more securely, efficiently, quickly and intelligently.

THT Token
The Thought (THT) Token allows for nodes on the network to exchange assets and Concepts (snippets of smart contract code stored on the blockchain)
Active Data Transformation Fabric

Pieces of Smart Data are processed in a widespread Active Data Transformation Fabric that allows them to make decisions and take actions based on their own internal logic. Each piece of smart data is secured individually, to allow value derived from the data to be shared while the original data is protected. Together, the Smart Data and Active Data Transformation Fabric enable Data Defined Computing, which merges processing and data into one.
